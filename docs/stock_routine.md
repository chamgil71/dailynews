# 📊 주식 시황 브리핑 루틴 v5

## 개요

매 거래일 오후 9:25 KST에 자동 실행.
역할: 시장 데이터 수집 → LLM 분석 → MD 저장 → Git push
배포(이메일·Notion·HTML)는 GitHub Actions stock_build.yml이 처리.

---

## Step 1: 시장 지수·환율·금리·유가 수집

다음 티커 목록의 현재가와 등락률을 수집하라.
**MCP 도구: `UsStockInfo.get_stock_info`**

수집 대상 (한 번에 하나씩, 순서대로):

- `^KS11`  → 코스피
- `^KQ11`  → 코스닥
- `KRW=X`  → 원/달러 환율
- `^GSPC`  → S&P 500
- `^IXIC`  → 나스닥
- `^DJI`   → 다우존스
- `^TNX`   → 미 10년물 국채금리 (단위: %, bp 변동 계산)
- `CL=F`   → WTI 유가

수집 결과 형식:

```json
{
  "kospi":    { "close": 2650.3,  "change_pct": 0.82 },
  "kosdaq":   { "close": 850.1,   "change_pct": -0.31 },
  "usd_krw":  { "close": 1380.5,  "change_pct": -0.15 },
  "sp500":    { "close": 5300.2,  "change_pct": 0.45 },
  "nasdaq":   { "close": 18500.0, "change_pct": 0.62 },
  "dow":      { "close": 40200.1, "change_pct": 0.28 },
  "us10y":    { "value": 4.32,    "change_bp": -3.5 },
  "wti":      { "close": 78.4,    "change_pct": -0.71 }
}
```

## Step 4: LLM 분석

수집한 데이터를 종합해 아래 항목을 분석하라:

1. **핵심 요약 (3줄)**: 오늘 시장의 핵심 움직임을 3문장으로 요약
2. **핵심 키워드 TOP 5**: 오늘 시장을 설명하는 키워드 5개 (예: `#반도체 #달러강세 #금리하락`)
3. **시장 온도계**: 🔴 과열 / 🟠 상승 / 🟡 중립 / 🟢 하락 / 🔵 침체 중 하나 선택
4. **온도계 근거**: 선택 이유를 1~2문장으로 설명
5. **주목 섹터**: 오늘 특히 주목할 섹터와 이유 (1~2문장)
6. **리스크 요인**: 단기 리스크 요인 1~2개

- 실행일(오늘 날짜)이 아니다. 예: 토요일(6/13) 실행 시 금요일(6/12) 데이터 → 파일명 `stock_2026-06-12.md`

## Step 6: GitHub에 직접 Push (mcp__github__push_files)

**⚠️ 웹 루틴은 세션 브랜치에서 실행되므로 `git push origin main`이 불가. 반드시 MCP 도구를 사용해 main에 직접 커밋한다.**

**⚠️ 날짜는 반드시 Step 1 거래일(trading_date) 사용. 실행일(오늘) 아님.**

```
도구: mcp__github__push_files
파라미터:
  owner: chamgil71
  repo: dailynews
  branch: main
  message: "📊 주식 시황 {거래일}"        ← Step 1의 trading_date
  files: [{ path: "reports/stock/stock_{거래일}.md", content: <MD 전체 내용> }]
                                          ← 예: stock_2026-06-12.md (금요일이면 금요일 날짜)
```

push 성공 시 GitHub Actions `stock_build.yml`이 자동으로:

- HTML 빌드 및 GitHub Pages 배포
- 이메일 발송 (익일 08:00 KST, stock_send.yml)
- Notion 동기화
- 히스토리 업데이트

**Step 6 완료 후 루틴 종료.**

> Notion·이메일·텔레그램은 GitHub Actions에서 처리하므로 루틴에서는 실행하지 않는다.
