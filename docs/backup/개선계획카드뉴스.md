**수정 방법:** `scripts/build_site.py` 파일의 HTML 변환 부분에 `toc`를 추가하세요.
```python
# scripts/build_site.py의 build_report_page 함수 수정

    raw,
    extras=["tables", "fenced-code-blocks", "strike", "cuddled-lists", "toc", "header-ids"]
)
```
이렇게 하면 마크다운의 `##` 헤더들에 자동으로 ID가 부여됩니다. 그런 다음 AI 프롬프트에서 상단에 목차를 스스로 작성하도록 지시할 수 있습니다.

### 3. 이메일 본문에 접기/펴기 (Collapsible List) 적용
**이메일에서는 권장하지 않습니다.**
HTML의 `<details>`와 `<summary>` 태그를 이용한 접기/펴기 기능은 웹(웹브라우저)에서는 완벽하게 동작하지만, **Gmail, Outlook, 네이버 메일 등 대부분의 이메일 클라이언트에서는 보안 및 렌더링 엔진의 한계로 해당 태그를 무시하거나 항상 펼쳐진 상태로 강제 노출**시킵니다. 

이메일 본문이 지저분해지는 것을 방지하려면, 전체 기사 목록은 이메일에서 아예 빼거나 간소화하고 **"웹에서 전체 기사 보기" 버튼을 넣는 방식**이 훨씬 깔끔합니다.

### 2. 파이썬 연동 로직 제안 (`scripts/build_site.py` 수정 방향)

`config/prompts.py`에서 정의된 "핵심 이슈 TOP 3"와 "주목할 트렌드" 텍스트를 파싱하여 위 HTML 구조에 끼워 넣는 로직이 필요합니다.

**추가해야 할 파싱 함수 예시:**

```python
def generate_card_news_html(analysis_text):
    """
    분석 텍스트에서 이슈 3개, 키워드 3개를 추출해 카드뉴스 HTML을 생성합니다.
    """
    # 1. 정규표현식 등으로 이슈와 키워드 파트를 분리 (현재 prompt 구조 활용)
    # 이슈 파트 추출 (1. **제목** — 내용)
    issues = re.findall(r'\d+\.\s+\*\*(.+?)\*\*\s*[—-]\s*(.+)', analysis_text)
    # 키워드 파트 추출 (- **키워드**: 내용)
    keywords = re.findall(r'-\s+\*\*(.+?)\*\*[:：]\s*(.+)', analysis_text)

    card_html = '<div class="card-news-container">'
    
    # 이슈 카드 생성 (최대 3개)
    icons = ["🔥", "📢", "💡"]
    for i, (title, content) in enumerate(issues[:3]):
        card_html += f"""
        <div class="news-card type-issue">
            <div class="card-header"><div class="icon">{icons[i]}</div><h3>{title}</h3></div>
            <div class="card-body">{content}</div>
            <div class="card-footer">핵심 이슈 TOP 3</div>
        </div>"""

    # 키워드 카드 생성 (최대 3개)
    k_icons = ["🏷️", "📊", "🎯"]
    for i, (title, content) in enumerate(keywords[:3]):
        card_html += f"""
        <div class="news-card type-keyword">
            <div class="card-header"><div class="icon">{k_icons[i]}</div><h3>#{title}</h3></div>
            <div class="card-body">{content}</div>
            <div class="card-footer">주목할 트렌드</div>
        </div>"""
        
    card_html += '</div>'
    return card_html
```

scripts/
├── build_site.py            ← 테마 로더 + MD 파싱만 담당

    ├── __init__.py
    ├── classic.py           ← 현재 디자인 (기본값)
