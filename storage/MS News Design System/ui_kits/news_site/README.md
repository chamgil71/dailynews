# MS News — Website UI Kit

Recreation of the public **AI News Daily** website (the only product surface
in the source repo). The original is a static site generated by
`scripts/build_site.py` and deployed to GitHub Pages / Vercel.

## What's here

- **`index.html`** — full interactive prototype. Three view modes wired up:
  - **Latest** (matches `publish/index.html` — one long report card)
  - **App** (matches `publish/app.html` — sidebar + tabs + filters + search)
  - **Archive** (matches `publish/archive.html` — list of all days)
- **`data.js`** — three sample reports lifted from the real site.
- Components (load order matters — components.jsx exports to `window`):
  - `Header.jsx`, `Footer.jsx`
  - `Badge.jsx`, `Card.jsx`, `Blockquote.jsx`
  - `NewsItem.jsx`, `NewsList.jsx`
  - `Sidebar.jsx` (search + stats + date list)
  - `Tabs.jsx` (all / EN / KO language filter)
  - `ReportView.jsx` (long-form), `AppView.jsx`, `ArchiveView.jsx`

## Source

`publish/index.html`, `publish/app.html`, `publish/archive.html` in
[chamgil71/dailynews](https://github.com/chamgil71/dailynews). CSS values are
inlined into the kit; tokens live in `../../colors_and_type.css`.

## Skipped on purpose

- No login / no accounts — the real product has none.
- The email-subscription card uses a placeholder `forms.gle` URL (matches source).
- No mobile app surface — the product is web + email only.
